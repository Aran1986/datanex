📦 فایل دانلود شده: DataNex-Auto-Launcher.zip

═══════════════════════════════════════════════════════════════════

✅ مرحله 1: Extract کردن
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. دانلود کن DataNex-Auto-Launcher.zip
2. Extract کن (Right click → Extract All)
3. کپی کن 2 فایل .bat رو به پوشه پروژه:
   
   📂 C:\Users\aran\Desktop\DataNex\
   ├── 📄 START_DATANEX.bat    ← کپی کن اینجا
   └── 📄 STOP_DATANEX.bat     ← کپی کن اینجا

═══════════════════════════════════════════════════════════════════

✅ مرحله 2: نصب پیش‌نیازها (فقط یکبار!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

نصب کن این 3 تا نرم‌افزار (اگه نداری):

🐍 Python 3.11+
   https://www.python.org/downloads/
   ⚠️ تیک بزن "Add Python to PATH"

📦 Node.js 18+ (LTS)
   https://nodejs.org/

🐋 Docker Desktop
   https://www.docker.com/products/docker-desktop/
   بعد از نصب، باز کن Docker Desktop

═══════════════════════════════════════════════════════════════════

🚀 مرحله 3: اجرا!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. اطمینان حاصل کن Docker Desktop اجرا شده

2. دابل کلیک کن روی:
   📄 START_DATANEX.bat

3. صبر کن 10-15 ثانیه

4. مرورگر خودکار باز میشه! 🎉

═══════════════════════════════════════════════════════════════════

✅ چی باید ببینی؟
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📺 دو تا terminal window باز میشه:
   • DataNex Backend  (API Server)
   • DataNex Frontend (UI)

🌐 مرورگر باز میشه با:
   http://localhost:3000

═══════════════════════════════════════════════════════════════════

🛑 خاموش کردن
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

دابل کلیک کن روی:
📄 STOP_DATANEX.bat

❓ سوالات متداول
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: اولین بار طول می‌کشه؟
A: بله، نصب dependencies حدود 5-10 دقیقه
   دفعات بعدی خیلی سریع‌تره! (10-15 ثانیه)

Q: دو تا terminal رو ببندم؟
A: نه! باید باز بمونن تا وقتی با DataNex کار می‌کنی

Q: لینوکس لازمه؟
A: نه! روی Windows کار می‌کنه

Q: چطوری بفهمم Docker اجرا شده؟
A: آیکون Docker در system tray سبز باشه
